141.环形链表

Author: Lin Date: March 7, 2024  22:28:26 Category: leetcode-hot100

问题：

给你一个链表的头节点 head ，判断链表中是否有环。

如果链表中有某个节点，可以通过连续跟踪 next 指针再次到达，则链表中存在环。 为了表示给定链表中的环，评测系统内部使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。注意：pos 不作为参数进行传递 。仅仅是为了标识链表的实际情况。

如果链表中存在环 ，则返回 true 。 否则，返回 false 。

思考：

  • 环问题：快慢指针

    • 有环：就类似龟兔赛跑，快指针最终多绕几圈就可以与慢指针会和
    • 无环：快指针一定会先到达null值

  • 循环临界条件：

    • fast == null，fast == slow
    • 额外条件： fast.next == null ，避免出现空指针异常
