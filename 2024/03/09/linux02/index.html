linux常用命令02 — 文档编辑

Author: Lin Date: March 9, 2024  19:06:49 Category: 测试 —— Linux基本操作

1.cat — 查看文件内容

  • cat 参数 文件名
  • 适合查看内容较少的纯文本文件
  • 参数：
    • -n：显示行数
    • --version：查看版本信息
  • cat a.txt


2.more — 分页显示文本文件内容

  • more 文件名
    • 该命令可以把文本内容一页一页显示在终端界面上
      • 按下enter，向下一行
      • 按下空格键，向下一页


3.grep — 在指定文件中查找指定的字符串

  • grep 参数 文件名

    • 参数：

      • 指定字符串：grep include test.c
      • 搭配正则表达式
      • -n：显示行号
      • -v：查找不包含指定内容的行
      • -l：忽略大小写


4.echo — 输出字符串或提取变量值



5.>、>> — 重定向符



6.管道 | — 将一个命令的输出作为另一个命令的输入

  • 管道的作用：

    • 过滤数据：ls -al | grep a 将ls -al的输出作为grep a的输入

    • 排序数据：ls -l | sort -r

    • 合并数据：cat a.txt test.c | grep include



7. vi — 文本编辑器

  • 功能：vi文本编辑器

  • 三种模式：

    • 命令模式
    • 插入模式：i
    • 末行模式：esc

  • 退出（末行模式下）：

    • :q! => 不保存退出
    • :wq => 保存退出
